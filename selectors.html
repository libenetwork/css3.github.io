СSS-селектори

Вибір шрифтів то кольорів є важливим етапом у розробці веб-інтерфейсу. Рекомендоване використання не більше трьох шрифтів і їх накреслень, але вони мають підкреслювати дизайн, передавати його настрій. Шрифт може підкреслювати деякі його особливості. Однак основний текст має гарно читатися, тому не рекомендується використання для того декоративних шрифтів. Самі вони мають гармоніювати між собою

Селектори є важливим елементом роботи із CSS. Без них неможливо обирати елементи в HTML-документі і вказувати його властивості.

CSS дозволяє налаштовувати будь який HTML-тег, за допомогою вбудованих селекторів. 

body{
background-color: red;
}
h1{
font-family: Consolas;
}

Копіювати

Окрім цього HTML дає можливість іменувати теги за допомогою class та id. Для вибору сlass-тегів необхідно додати перед назвою класу «.», для id «#»

<head>
  <style>
    body{
      background-color: black;
    }
    .hello{
       font-family: Montserrat;
       color: white;
    }
    #hello{
       font-family: Roboto;
    }
  </style>
</head>
<body>
  <p class = "hello"> Hello World!</p>
  <p id = "hello">
            Привіт, Світе! </p>
</body>

Копіювати

Одразу можна обирати декілька тегів, тоді правило буде працювати для всіх перерахованих. Для цього використовується «,». Це допоможе скоротити кількість правил і прискорити завантаження.

<head>
  <style>
    body{
      background-color: black;
    }
    .hello, #hello{
       font-family: Montserrat;
       color: white;
    }
    </style>
</head>
<body>
  <p class = "hello"> Hello World!</p>
  <p id = "hello">
            Привіт, Світе! </p>
</body>

Копіювати

CSS також підтримує систему наслідувань. Вони реалізовуються за допомогою «>» та просто перерахування символів. Останній селектор вказує на всі дочірні елементи, навіть якщо вони знаходяться в середині іншого контейнера. В першому мова йде про ті що безпосередньо в цьому контейнері.

<head>
  <style>
    .hello p{
      background-color: black;
    }
    #hello > p {
       font-family: Montserrat;
       color: white;
    }
    </style>
</head>
<body>
  <div class = "hello">
    <p> Hello World!</p>
    <div id = "hello">
            <p> Привіт, Світе!</p>
    </div>
  </div>
</body>

Копіювати

Також в СSS існують селектори, які застосовуються тільки у випадку, якщо спрацьовує певна властивість. Вони позначаються символом «:». Наприклад :hover вказує, що на об'єкт наведено мишею, а :active, що на посилання було нажато

.hello:hover{
   background-color: black;
}
