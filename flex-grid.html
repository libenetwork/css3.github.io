Cітка Flex

Створення модульної сітки вкрай важливий етап у створенні веб-інтерфейсів. За допомоги нею можна визначити відносини між елементами інтерфейсу, підтримувати певні пропорції в інтерфейсі. Cітка допомагає впорядкувати елементи інтерфейса, зробити вигляд інтерфейсу більш приємним для перегляду. Сітка визначає єдиний стиль оформлення. Дає відчуття скомпонованості Як не складно здогадатися основний елемент модульної сітки є модель. Розбиття екрану на модулі: колонки чи рядки, дозволяє завчасно продумати як виглядатиме веб-інтерфейс. СSS має два основних способи верстання модульної сітки: Flex та Grid.

Flex-контейнер

Flex-сітка являє собою одномірну сітку. Тобто простір за її допомогою ділиться на колонки, а всі елементи розташовуються вздовж однієї вісі

Контейнеру Flex можна задати вісь вздовж якої потрібно розміщувати елементи

.container {
  flex-direction: row | row-reverse | column | column-reverse;
}

Копіювати

За замовчуванням усі flex елементи намагатимуться вмістити в один рядок. Ви можете змінити це та дозволити обтікання елементів за потреби за допомогою цієї властивості.

.container {
  flex-wrap: nowrap | wrap | wrap-reverse;
}
